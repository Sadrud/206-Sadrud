Обязательно она найдет какой-\/либо путь (необязательно эйлеров). \textbackslash{}И, в итоге, имея путь, функция опять проходит по всем вершинам этого пути, и вычитает ребро из матрицы смежности, и проделывает так, пока не пройдет путь. \textbackslash{}Если получилась нулевая матрица, по найденный путь эйлеров, иначе нет.
