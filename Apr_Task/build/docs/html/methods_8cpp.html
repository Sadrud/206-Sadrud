Основной метод - нахождение эйлерового пути в графе \Функция работает с матрицей смежности графа. \Сначала она проверяет, существует ли вообще эйлеров путь (также и эйлеров цикл). \Далее она выбирает произвольную вершину, и начиная с нее уже начинает бегать по графу. Обязательно она найдет какой-либо путь (необязательно эйлеров). \И, в итоге, имея путь, функция опять проходит по всем вершинам этого пути, и вычитает ребро из матрицы смежности, и проделывает так, пока не пройдет путь. \Если получилась нулевая матрица, по найденный путь эйлеров, иначе нет.

Аргументы
[in]n,gКоличесвто вершин и матрица смежности
Возвращает
Возвращает вектор, содержащий эйлеров путь
